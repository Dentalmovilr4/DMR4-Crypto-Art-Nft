# 🎨 DMR4 Crypto Art Nft Gallery - Genesis Collection

Bienvenido a la galería oficial de **DMR4 Crypto Art**. Este proyecto es una exhibición digital de activos únicos (NFTs) que fusionan el arte tradicional, el estudio anatómico y la ilustración digital con la tecnología Web3.

## 🚀 Sobre el Proyecto

Esta galería ha sido desarrollada íntegramente desde un entorno móvil utilizando **Termux** y **Acode**, demostrando el poder del desarrollo descentralizado y portátil. Cada pieza representa una visión única del arte digital y la cultura técnica.

### Características Principales:
- **Automatización:** El `index.html` carga dinámicamente los metadatos desde archivos JSON.
- **Estética:** Diseño con estilo oscuro y acentos neón (Cian/Rosa).
- **Web3 Ready:** Estructura de metadata compatible con estándares de marketplaces como OpenSea.

## 📁 Estructura del Repositorio

- `/assets/nfts/`: Contiene las piezas de arte originales en formato `.jpg` y `.png`.
- `/metadata/`: Archivos JSON con la información técnica y descripción de cada NFT.
- `index.html`: Interfaz de usuario automatizada con JavaScript.

## 🛠️ Tecnologías Usadas

- **HTML5 / CSS3** (Estilo Neón Personalizado)
- **JavaScript** (Carga asíncrona de archivos JSON)
- **GitHub Pages** (Hosting de la galería)
- **Entorno de desarrollo:** Oppo A57 vía Termux.
